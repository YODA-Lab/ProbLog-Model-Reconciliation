# ProbLog MRP
Repository for the code and data associated with the NeurIPS 2025 paper: "Model Reconciliation via Cost-Optimal Explanations in Probabilistic Logic Programming."

## Includes:

1. Appendix
2. Cost Estimation
3. Computational Results

### 1. Appendix:

Additional information referenced in the main paper.

### 2. Cost Estimation:

Details of a human-subject study where participants interacted with an AI assistant named  *Blitzcrank*. The study supports the cost estimation results presented in Section 6.1 and enables reproducibility.

### 3. Computational Results:

Implementation of both generic and optimized search algorithms used in the experiments reported in Section 6.2.

#### Code prerequisites:

- Python == 3.7+
